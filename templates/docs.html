Complete guide to Web Optimizr features and tools

Overview

Web Optimizr is a privacy-focused toolkit for developers. It runs entirely in-memory (on the server-side for the web app, or locally for the desktop app), ensuring your files are never saved to a disk.

üöÄ Image Optimizer

Compress and resize images for the web.

  • Input Formats: JPG, PNG, JPEG.
  • Output Format: WebP (Modern, high compression).
  • Customization: Set max width (files resize proportionally) and compression quality (1-100).

Reduce file size of your source code for production.

Removes whitespace and comments using jsmin.

Installation Guide

Linux (Recommended: Binary)

1. Download the pre-built binary:

Download WebOptimizer_Linux

2. Open your terminal in the download folder.

3. Make the file executable and run it:

chmod +x WebOptimizer_Linux
./WebOptimizer_Linux

Linux (Alternative: Source)

If the binary doesn't work, you can run from source:

# Install Python and Tkinter
sudo apt install python3 python3-pip python3-tk

# Clone or download the source code
# (Assuming you have the files)

# Install dependencies
pip install -r requirements.txt

# Run the app
python3 desktop_app.py

Windows (Portable Exe)

1. Download the standalone executable:

Download WebOptimizr.exe

2. Double-click to run. No installation required.
